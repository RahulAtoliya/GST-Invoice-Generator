# GST Invoice Generator

A complete Next.js GST invoice generator for Indian businesses. It includes a public landing page, local login/signup, profile-based default seller details, invoice creation, validation, automatic GST calculations, local invoice history, preview, and downloadable A4 PDFs.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Included

- Next.js App Router with TypeScript
- Tailwind CSS styling
- React Hook Form and Zod validation
- LocalStorage invoice management
- LocalStorage login/signup and profile defaults
- LocalStorage admin login with user management dashboard
- jsPDF invoice download
- Visitor landing page, dashboard, invoice history, create, edit, preview, delete, and re-download flows
